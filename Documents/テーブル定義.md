# テーブル定義

本アプリケーションはデータベースを使用しません（N/A）。

データはすべて Claude Desktop / Claude Code が管理する JSON ファイルとして永続化されます。以下に、本アプリが読み書きする永続化ファイルの構造を「擬似テーブル」として記載します。

## ~/.claude.json（Claude Code グローバル設定）

トップレベルに多数のキーを持つ巨大ファイル。本アプリは `mcpServers` セクションのみを操作する。

| カラム | 型 | 説明 |
| --- | --- | --- |
| mcpServers | object（map） | キー = サーバー名、値 = MCPServerConfig。本アプリの操作対象。 |
| projects | object | プロジェクト別設定（`mcpServers` を含むが本アプリは対象外）。保持のみ。 |
| その他多数 | 各種 | 起動カウンタ・キャッシュ・アカウント情報など。本アプリは保持のみ（変更しない）。 |

## ~/.claude-disabled-mcp.json（Claude Code 無効 MCP サイドカー）

無効化した Claude Code MCP の退避先。ホーム直下。空になると削除される。

| カラム | 型 | 説明 |
| --- | --- | --- |
| mcpServers | object（map） | キー = サーバー名、値 = MCPServerConfig。 |

## MCPServerConfig（mcpServers の値）

| カラム | 型 | 説明 |
| --- | --- | --- |
| command | string | 実行コマンド。 |
| args | string[]（任意） | コマンド引数。 |
| env | object（任意） | 環境変数。 |
| type | string（任意） | エントリ種別（例 `"stdio"`）。Claude Code 用。移動時に保持する。 |
| disabled | boolean（任意） | 無効フラグ（Claude Desktop の一部設定で使用）。 |
| （その他） | unknown | 未知フィールドも不透明に保持する。 |

## claude_desktop_config.json / claude_desktop_config_disabled.json（Claude Desktop 設定）

OS ごとの Claude Desktop 設定ディレクトリに配置。無効化したサーバーは `_disabled` 側に移動する。

| カラム | 型 | 説明 |
| --- | --- | --- |
| mcpServers | object（map） | キー = サーバー名、値 = MCPServerConfig。 |
| （その他） | unknown | その他のキーは保持する。 |

## ~/.claude 配下 クリーンアップ対象ディレクトリ

DB ではないが、クリーンアップ機能が扱うディレクトリ一覧を参考として記載する。詳細は [システム仕様.md](システム仕様.md) のクリーンアップ候補表を参照。

デフォルトはすべてチェック OFF。

| ディレクトリ | 区分 |
| --- | --- |
| projects | 履歴・メモリー（個別選択可） |
| file-history | 履歴 |
| shell-snapshots | 一時 |
| cache | キャッシュ |
| debug | ログ |
| sessions | 一時 |
| session-env | 一時 |
| tasks | 一時 |
| backups | バックアップ |
